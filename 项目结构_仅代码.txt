一、项目概述
===========
本项目是一个基于深度学习的掌纹识别系统，主要用于提取和识别掌纹中的主要线条。项目使用PyTorch框架，采用UNet和ResUNet模型进行图像分割任务。

三、核心模块说明
===============

3.1 数据处理模块
----------------

3.1.1 src/data_loader.py
功能：数据集加载和预处理
主要类：
  - PLSUDataset: 主要数据集类，继承自torch.utils.data.Dataset
特性：
  - 支持数据增强（Albumentations库）
  - 支持线条提取（简单方法和复杂方法）
  - 支持训练/验证/测试数据集划分
  - 自动处理图像和掩码的对应关系

3.1.2 src/preprocessor.py
功能：图像预处理
主要操作：
  - 图像归一化
  - 图像大小调整
  - 数据增强（旋转、翻转、缩放等）
  - 颜色空间转换

3.1.3 src/line_filter.py
功能：线条提取和过滤
主要算法：
  - K-means聚类：用于线条分离
  - 形态学操作：膨胀、腐蚀、开运算、闭运算
  - 骨架化：提取线条中心线
  - 轮廓检测：识别线条边界

3.2 模型架构模块
----------------

3.2.1 src/models/unet.py
功能：UNet分割模型
架构特点：
  - 编码器-解码器结构
  - 跳跃连接保留特征
  - 适合医学图像分割
主要组件：
  - DoubleConv: 双卷积块
  - Down: 下采样块
  - Up: 上采样块
  - OutConv: 输出卷积层

3.2.2 src/models/resunet.py
功能：ResUNet模型（带残差连接）
架构特点：
  - 在UNet基础上加入残差连接
  - 更深的网络结构
  - 更好的梯度传播
  - 适合复杂场景的分割任务

3.3 训练与评估模块
------------------

3.3.1 src/trainer.py
功能：模型训练逻辑
主要功能：
  - 损失函数计算（Dice Loss、BCE Loss、Focal Loss）
  - 优化器配置（Adam、SGD等）
  - 学习率调度（ReduceLROnPlateau）
  - 模型检查点保存
  - 训练日志记录

3.3.2 src/evaluator.py
功能：模型评估
评估指标：
  - IoU (Intersection over Union)
  - Precision (精确率)
  - Recall (召回率)
  - F1 Score
  - Dice Coefficient

3.3.3 src/predictor.py
功能：模型预测
输出内容：
  - 分割掩码
  - 线条提取结果
  - 可视化图像
  - 预测置信度

3.4 配置与工具模块
------------------

3.4.1 src/config.py
功能：全局配置参数
配置内容：
  - 模型参数（输入大小、通道数、类别数）
  - 训练参数（批次大小、学习率、训练轮数）
  - 数据路径
  - 设备配置（CPU/GPU）

3.4.2 src/utils.py
功能：通用工具函数
工具函数：
  - 图像处理函数
  - 文件操作函数
  - 日志记录函数
  - 可视化函数

五、数据集说明
==============

5.1 数据集概述
--------------
数据集名称：PLSU (Palm Line Segmentation Universe)
数据集规模：约280对图像-掩码
图像格式：JPG (RGB)
掩码格式：PNG (Grayscale)

5.2 图像目录 (PLSU/img/)
-------------------------
图像数量：约280张
图像格式：JPG
命名规则：image{数字}.jpg
说明：包含掌纹原始图像，用于模型训练和测试

5.3 掩码目录 (PLSU/Mask/)
-------------------------
掩码数量：约280张
掩码格式：PNG
命名规则：image{数字}.png
说明：包含掌纹线条标注掩码，用于监督学习

5.4 数据对应关系
----------------
- 每张图像都有对应的掩码
- 图像和掩码通过文件名对应
- 掩码用于标注掌纹线条的位置
- 支持多线条标注

九、项目使用流程
==================

9.1 环境配置
------------
步骤1：创建虚拟环境（可选）
  - 使用conda: conda create -n palm python=3.8
  - 或使用venv: python -m venv venv

步骤2：激活虚拟环境
  - Conda: conda activate palm
  - venv: source venv/bin/activate (Linux/Mac)
  - venv: .\venv\Scripts\activate (Windows)

步骤3：安装依赖
  - 使用pip: pip install -r requirements.txt

9.2 数据准备
------------
步骤1：准备数据集
  - 将掌纹图像放入 PLSU/img/ 目录
  - 将标注掩码放入 PLSU/Mask/ 目录

步骤2：检查数据对应关系
  - 确保每张图像都有对应的掩码
  - 文件名必须一致（除扩展名外）

9.3 模型训练
------------
步骤1：配置训练参数
  - 编辑配置文件（src/config.py）
  - 设置模型参数、训练参数等

步骤2：运行训练脚本
  - python scripts/train_final.py

步骤3：监控训练过程
  - 查看训练日志
  - 观察损失曲线
  - 检查验证指标

步骤4：保存最佳模型
  - 模型自动保存到 checkpoints/ 目录
  - 选择验证集上表现最好的模型

9.4 模型评估
------------
步骤1：运行评估脚本
  - python scripts/evaluate.py

步骤2：查看评估结果
  - 查看评估指标（IoU、Precision、Recall、F1）
  - 分析模型性能

9.5 模型预测
------------
步骤1：运行预测脚本
  - python scripts/predict_final.py

步骤2：查看预测结果
  - 查看 predictions/ 目录中的对比图
  - 分析预测质量

十二、常见问题及解决方案
======================

12.1 内存不足
-------------
解决方案：
  - 减小批次大小
  - 使用梯度累积
  - 减小图像尺寸
  - 使用混合精度训练

12.2 训练速度慢
-------------
解决方案：
  - 使用GPU加速
  - 优化数据加载（增加num_workers）
  - 使用更快的存储设备
  - 减少数据增强操作

12.3 模型过拟合
-------------
解决方案：
  - 增加数据增强
  - 使用正则化（Dropout、L2正则化）
  - 减小模型复杂度
  - 增加训练数据

12.4 预测结果不理想
-----------------
解决方案：
  - 调整模型参数
  - 增加训练轮数
  - 使用预训练模型
  - 优化损失函数

12.5 导入错误
-----------
解决方案：
  - 检查Python路径设置
  - 确保所有依赖已安装
  - 检查虚拟环境是否激活
  - 重新安装问题包
